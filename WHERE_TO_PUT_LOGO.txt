║                   LOGO & FAVICON PLACEMENT GUIDE                     ║
╚══════════════════════════════════════════════════════════════════════╝

📁 PLACE ALL FILES IN THIS FOLDER:
   frontend/public/

📂 FOLDER STRUCTURE:
   gaglawyers/
   └── frontend/
       └── public/          ← PUT YOUR FILES HERE!
           ├── logo.png                  (Your main logo)
           ├── favicon.ico               (32x32 or 16x16)
           ├── favicon.svg               (SVG format)
           ├── favicon-16x16.png         (16x16 PNG)
           ├── favicon-32x32.png         (32x32 PNG)
           └── apple-touch-icon.png      (180x180 PNG)

═══════════════════════════════════════════════════════════════════════

📋 FILE SPECIFICATIONS:

1. logo.png
   ✓ Size: 200px width × 60px height (or maintain aspect ratio)
   ✓ Format: PNG with transparent background
   ✓ Colors: Should look good on navy blue background
   ✓ File size: Keep under 50KB

2. favicon.ico
   ✓ Size: 32×32px or 16×16px
   ✓ Format: ICO
   ✓ For older browsers

3. favicon.svg
   ✓ Format: SVG (scalable)
   ✓ For modern browsers

4. favicon-16x16.png
   ✓ Size: 16×16px
   ✓ Format: PNG

5. favicon-32x32.png
   ✓ Size: 32×32px
   ✓ Format: PNG

6. apple-touch-icon.png
   ✓ Size: 180×180px
   ✓ Format: PNG
   ✓ For iOS devices (home screen icon)

═══════════════════════════════════════════════════════════════════════

🛠️ HOW TO GENERATE FAVICONS:

Option 1: Use Online Generator (RECOMMENDED)
   1. Go to: https://realfavicongenerator.net/
   2. Upload your logo
   3. Download the generated package
   4. Extract and copy all files to frontend/public/

Option 2: Use Favicon.io
   1. Go to: https://favicon.io/
   2. Upload your logo or create from text
   3. Download the package
   4. Copy files to frontend/public/

═══════════════════════════════════════════════════════════════════════

✅ AFTER PLACING FILES:

1. Restart your development server:
   cd frontend
   npm run dev

2. Check these locations:
   ✓ Navbar (top left) - Logo should appear
   ✓ Footer (bottom) - Logo should appear
   ✓ Browser tab - Favicon should appear
   ✓ Bookmarks - Favicon should appear

3. If logo doesn't appear:
   ✓ Check file name is exactly "logo.png" (lowercase)
   ✓ Check file is in frontend/public/ folder
   ✓ Clear browser cache (Ctrl+Shift+Delete)
   ✓ Hard refresh page (Ctrl+F5)

═══════════════════════════════════════════════════════════════════════

💡 TIPS:

• Logo should have transparent background
• Logo should be optimized (use tinypng.com)
• Test logo on navy background before uploading
• Keep file sizes small for fast loading
• Use 2x resolution for retina displays

❓ NEED HELP?

If logo doesn't show:
• File must be named exactly: logo.png
• File must be in: frontend/public/
• Clear browser cache and hard refresh
• Check browser console for errors

If favicon doesn't show:
• Clear browser cache completely
• Close and reopen browser
• Check all favicon files are present
• Wait a few minutes (browsers cache favicons)
