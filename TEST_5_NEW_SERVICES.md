## 🐛 Troubleshooting

### "Service Not Found" Error
**Solution:**
1. Check URL spelling (all lowercase, hyphens not underscores)
2. Verify service exists in `servicesData.js`
3. Check dev server is running
4. Refresh browser (Ctrl+Shift+R)

### Widget Not Appearing
**Solution:**
1. Check you're on the EXACT page (slug must match perfectly)
2. Clear browser cache
3. Check `ServiceDetail.jsx` has the conditional widget code
4. Verify slug comparison: `slug === 'motor-accident-mact-lawyer-in-delhi'`

### Badge Not Showing (Supreme Court)
**Solution:**
1. Check `servicesData.js` has `hero.badge: 'Advocate on Record (AOR) Certified'`
2. Verify ServiceDetail.jsx has badge rendering code
3. Clear browser cache

### Table Broken (Writ page)
**Solution:**
1. Verify all table rows have exactly 3 cells
2. Check for missing commas in data structure
3. Ensure table wrapper has `overflow-x-auto`

### Emergency Card Wrong Color
**Solution:**
1. Should be `bg-red-600`, not any other red shade
2. Check conditional rendering: `slug === 'sexual-harassment-section-498a-lawyer-in-delhi'`
3. Verify hex code is correct
