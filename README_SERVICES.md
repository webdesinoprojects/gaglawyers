## 🔧 Technical Architecture

### How It Works:

```
User visits URL
    ↓
/bail-lawyer-in-delhi
    ↓
App.jsx routes to ServiceDetail component
    ↓
ServiceDetail extracts slug from URL
    ↓
getServiceBySlug('bail-lawyer-in-delhi')
    ↓
Returns service data from servicesData.js
    ↓
ServiceDetail renders all sections dynamically
    ↓
Beautiful page displayed!
```
